# 每天学几个有用的Vim命令

## 显示行号
- 临时显示行号
```bash
:set number
```
- 全局永久显示行号
```bash
vim ~/.vimrc

set nu
```
## 多行添加/取消注释
  按Ctrl+v 进入Visual 模式，光标向下选取多行
  按a，在光标后插入内容，如#; 按ESC退出，则多行被添加注释
  如果需要取消注释，多行选择注释#, 然后按x 删除，则完成取消

## 快速到达文件末尾
```bash
shift +g
```
## 快速到达文件开头
```bash
gg
```
## 快速到达当前行头
```bash
^
```
## 按单词向后移动光标
```bash
w
```
